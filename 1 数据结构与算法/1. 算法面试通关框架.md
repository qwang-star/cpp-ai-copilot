# 算法面试通关框架

## 一、核心方法论

### 1.1 刷题三阶段

| 阶段 | 目标 | 题量 | 重点 |
|------|------|------|------|
| **入门** | 掌握基础数据结构 | 50 题 | 数组、链表、栈、队列、哈希表、二叉树遍历 |
| **进阶** | 掌握核心算法思想 | 100 题 | 双指针、滑动窗口、二分、DP、回溯、贪心 |
| **冲刺** | 按公司刷高频题 | 50 题 | 目标公司近 6 个月高频题 |

### 1.2 做题四步法

1. **审题** → 确认输入输出、边界条件、数据规模 → 推算法复杂度上限
2. **暴力解** → 先用最简单的思路写出来（即使 $O(n^2)$）
3. **优化** → 找重复计算 / 冗余遍历 → 换数据结构或算法
4. **边界测试** → 空输入、单元素、极大值、重复值

### 1.3 复杂度速查表

| 数据规模 n | 允许的最高复杂度 | 常见算法 |
|------------|-----------------|----------|
| n ≤ 10 | $O(n!)$, $O(2^n)$ | 回溯、排列组合 |
| n ≤ 20 | $O(2^n)$ | 状态压缩 DP |
| n ≤ 100 | $O(n^3)$ | Floyd、区间 DP |
| n ≤ 10³ | $O(n^2)$ | DP、双层循环 |
| n ≤ 10⁵ | $O(n\log n)$ | 排序、二分、堆 |
| n ≤ 10⁶ | $O(n)$ | 双指针、哈希、前缀和 |
| n > 10⁶ | $O(\log n)$ 或 $O(1)$ | 二分、数学公式 |

## 二、数据结构速查

### 2.1 数组 & 字符串

**核心操作**：遍历、反转、双指针

```cpp
// 双指针 - 反转数组
void reverse(vector<int>& nums, int l, int r) {
    while (l < r) swap(nums[l++], nums[r--]);
}

// 双指针 - 快慢指针找中点
ListNode* middle(ListNode* head) {
    ListNode *slow = head, *fast = head;
    while (fast && fast->next) {
        slow = slow->next;
        fast = fast->next->next;
    }
    return slow;
}
```

> 高频题型：双指针、滑动窗口、前缀和、原地修改

### 2.2 哈希表

**核心**：空间换时间，$O(1)$ 查找

```cpp
unordered_map<int, int> mp;    // key → value
unordered_set<int> st;          // 去重
```

> 高频题型：两数之和、字母异位词、最长连续序列、LRU Cache

### 2.3 栈 & 队列

```cpp
stack<int> stk;          // LIFO
queue<int> q;            // FIFO
deque<int> dq;           // 双端队列
priority_queue<int> pq;  // 最大堆（默认）
priority_queue<int, vector<int>, greater<int>> minHeap;  // 最小堆
```

> 高频题型：有效括号、单调栈（下一个更大元素）、接雨水、滑动窗口最大值

### 2.4 二叉树

```cpp
// 三种遍历（递归 + 迭代都要会）
void preorder(TreeNode* root);   // 根左右
void inorder(TreeNode* root);    // 左根右  
void postorder(TreeNode* root);  // 左右根
void levelOrder(TreeNode* root); // 层序 BFS
```

**面试必背**：
- BFS = 队列；DFS = 栈/递归
- 二叉搜索树（BST）：左 < 根 < 右，中序遍历有序
- 平衡二叉树（AVL / 红黑树）：高度差 ≤ 1

> 高频题型：遍历变体、LCA（最近公共祖先）、路径和、对称/翻转、序列化反序列化

### 2.5 堆

```cpp
// Top K 问题 → 最小堆（保留大的 K 个）
priority_queue<int, vector<int>, greater<int>> minHeap;
for (int x : nums) {
    minHeap.push(x);
    if (minHeap.size() > k) minHeap.pop();
}
```

> 高频题型：Top K、数据流中位数、合并 K 个有序链表、任务调度

### 2.6 图

| 遍历 | 结构 | 用途 |
|------|------|------|
| DFS | 栈/递归 | 连通性、环检测、拓扑排序 |
| BFS | 队列 | 最短路径（无权图）、层序遍历 |

```cpp
// BFS 最短路径模板
queue<int> q; q.push(start);
vector<bool> vis(n, false); vis[start] = true;
int step = 0;
while (!q.empty()) {
    int sz = q.size();
    while (sz--) {
        int cur = q.front(); q.pop();
        if (cur == target) return step;
        for (int next : adj[cur]) {
            if (!vis[next]) {
                vis[next] = true;
                q.push(next);
            }
        }
    }
    step++;
}
```

> 高频题型：岛屿数量、课程表（拓扑排序）、克隆图、最短路径

## 三、核心算法模板

### 3.1 二分查找

```cpp
// 标准二分：找 target，返回下标
int binarySearch(vector<int>& nums, int target) {
    int l = 0, r = nums.size() - 1;
    while (l <= r) {
        int mid = l + (r - l) / 2;
        if (nums[mid] == target) return mid;
        else if (nums[mid] < target) l = mid + 1;
        else r = mid - 1;
    }
    return -1;
}

// 左边界：第一个 ≥ target
int lower_bound(vector<int>& nums, int target) {
    int l = 0, r = nums.size();
    while (l < r) {
        int mid = l + (r - l) / 2;
        if (nums[mid] >= target) r = mid;
        else l = mid + 1;
    }
    return l;
}
```

> 口诀：**标准二分 `l <= r` 带等号，边界二分 `l < r` 不带等号**

### 3.2 排序

| 算法 | 平均时间 | 最坏时间 | 空间 | 稳定性 | 一句话 |
|------|---------|---------|------|--------|--------|
| 快排 | $O(n\log n)$ | $O(n^2)$ | $O(\log n)$ | ❌ | 选 pivot，partition |
| 归并 | $O(n\log n)$ | $O(n\log n)$ | $O(n)$ | ✅ | 分治 + 合并有序数组 |
| 堆排 | $O(n\log n)$ | $O(n\log n)$ | $O(1)$ | ❌ | 建堆 + 依次取堆顶 |

> 面试常问：手写快排 partition、归并排序链表、海量数据外部排序用归并

### 3.3 动态规划 (DP)

**五步法**：
1. 定义 dp 含义（dp[i] 代表什么）
2. 找状态转移方程
3. 初始化 base case
4. 确定遍历顺序
5. 举例验证

```cpp
// 经典：0-1背包
// dp[i][w] = max(dp[i-1][w], dp[i-1][w-wt[i]] + val[i])
// 优化为一维：倒序遍历
for (int i = 0; i < n; i++)
    for (int w = cap; w >= wt[i]; w--)
        dp[w] = max(dp[w], dp[w - wt[i]] + val[i]);
```

> 高频题型：背包问题、打家劫舍、股票买卖、最长子序列/子串、编辑距离

### 3.4 贪心

**核心**：局部最优 → 全局最优（需证明）

```cpp
// 区间调度：按结束时间排序，选不重叠的最多区间
sort(intervals.begin(), intervals.end(),
     [](auto& a, auto& b) { return a[1] < b[1]; });
```

> 高频题型：跳跃游戏、加油站、分发糖果、会议室、用最少箭射气球

### 3.5 回溯

**核心**：决策树遍历 → 选择 → 递归 → 撤销

```cpp
// 全排列模板
void backtrack(vector<int>& nums, vector<int>& path, vector<bool>& used) {
    if (path.size() == nums.size()) {
        res.push_back(path);
        return;
    }
    for (int i = 0; i < nums.size(); i++) {
        if (used[i]) continue;
        used[i] = true;
        path.push_back(nums[i]);
        backtrack(nums, path, used);
        path.pop_back();
        used[i] = false;
    }
}
```

> 高频题型：全排列、子集、组合总和、N 皇后、单词搜索

### 3.6 滑动窗口

```cpp
// 可变窗口模板
int l = 0, r = 0;
while (r < s.size()) {
    char c = s[r]; r++;
    // ... 更新窗口内数据
    while (/* 窗口需要收缩 */) {
        char d = s[l]; l++;
        // ... 更新窗口内数据
    }
    // ... 更新答案
}
```

> 高频题型：无重复字符的最长子串、最小覆盖子串、字符串排列

### 3.7 前缀和

```cpp
// preSum[i] = nums[0] + ... + nums[i-1]
// 区间 [i, j] 的和 = preSum[j+1] - preSum[i]
vector<int> preSum(n + 1, 0);
for (int i = 0; i < n; i++)
    preSum[i + 1] = preSum[i] + nums[i];
```

> 高频题型：和为 K 的子数组、二维区域和检索、差分数组

## 四、面试踩坑清单

| 坑 | 示例 | 对策 |
|-----|------|------|
| **溢出** | `int mid = (l + r) / 2` → l+r 可能溢出 | 用 `l + (r - l) / 2` |
| **空指针** | 链表操作未判空 | 任何 `.next` 前先判 `!= nullptr` |
| **死循环** | 二分 `while (l < r)` 不收敛 | 检查 `mid` 偏向，必要时 `mid + 1` / `mid - 1` |
| **边界** | 数组越界、负数下标 | 谨慎用 `size()-1`，先判空 |
| **重复元素** | 排序去重忘判 `i > 0` | 去重时先判断重复条件 |
| **INT_MIN 取反** | `abs(INT_MIN)` 溢出 | 单独处理或转 `long long` |

## 五、面试脑内检查清单

每题写完都过一遍：

- [ ] 有没有指针为空的情况？
- [ ] 有没有数组越界？
- [ ] 有没有死循环？
- [ ] 有没有整数溢出？
- [ ] 输入为空、单元素、全相同的情况覆盖了吗？
- [ ] 时间复杂度是否符合题目数据规模？

> **记住：面试不是考你会不会，是考你犯不犯错。**
