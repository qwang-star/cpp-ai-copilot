# F02 备选项目库

## 1. 智能客服系统

### 项目是什么

基于企业 FAQ、订单系统、工单系统，为用户提供自动客服能力。

### 核心链路

```text
用户问题
  -> 意图识别
  -> RAG 知识库问答
  -> Tool Calling 查询订单 / 工单
  -> 无法解决转人工
  -> 生成客服摘要
```

### 亮点

- RAG。
- Tool Calling。
- 多轮对话。
- 转人工。
- 工单摘要。
- 权限控制。

### 面试难点

- 如何判断转人工。
- 如何保证订单查询权限。
- 如何减少客服幻觉。
- 如何统计用户满意度。

## 2. 简历 / 文档解析助手

### 项目是什么

上传简历或文档，自动抽取结构化信息并给出分析建议。

### 核心链路

```text
上传文件
  -> 文档解析 / OCR
  -> 文本清洗
  -> LLM 结构化抽取
  -> JSON Schema 校验
  -> 入库
  -> 生成建议
```

### 亮点

- 文件解析。
- OCR。
- 结构化输出。
- Prompt 版本管理。
- JSON Schema 校验。
- 异步任务。

### 面试难点

- 如何处理模型输出 JSON 不合法。
- 如何评估抽取准确率。
- 如何处理隐私数据。

## 3. 代码知识库问答

### 项目是什么

对代码仓库进行解析和索引，支持自然语言问代码。

### 核心链路

```text
拉取仓库
  -> 解析文件结构
  -> 按函数 / 类 / 文件切分
  -> 生成 embedding
  -> 用户提问
  -> 检索相关代码
  -> LLM 解释
```

### 亮点

- 代码切分。
- 代码语义检索。
- Git commit 信息。
- Tool Calling 查文件。
- 引用代码位置。

### 面试难点

- 代码如何 chunk。
- 如何处理跨文件调用。
- 如何避免模型编造代码行为。

## 4. 会议纪要助手

### 项目是什么

把会议音频转成文字，生成摘要、待办和决策。

### 核心链路

```text
上传音频
  -> ASR 转写
  -> 说话人分离
  -> 文本纠错
  -> 摘要
  -> 待办抽取
  -> 发送会议纪要
```

### 亮点

- ASR。
- 长文本摘要。
- 结构化抽取。
- 异步任务。
- 多模态。

## 5. 项目选择建议

秋招主项目优先：

```text
企业知识库问答系统
```

原因：

- 最贴近后端 AI 应用岗位。
- 能覆盖 RAG、MQ、Redis、MySQL、向量库、模型网关、安全、评测。

备选项目用来补充差异化。
