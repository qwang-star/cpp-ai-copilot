# B01 编程语言

## 1. Java

### 是什么

Java 是国内后端秋招最主流的语言之一，生态成熟，适合构建企业级后端系统。

### 为什么重要

大量后端岗位默认考 Java：

- 集合。
- JVM。
- 多线程。
- Spring。
- MySQL。
- Redis。
- MQ。

### AI 应用场景

Java 可以负责：

- 用户系统。
- 知识库系统。
- 文档管理。
- 会话管理。
- 模型网关。
- 权限和审计。
- 企业微服务集成。

### 必会知识点

- 基础类型和引用类型。
- 集合框架。
- HashMap。
- ConcurrentHashMap。
- 线程池。
- synchronized。
- volatile。
- AQS。
- JVM 内存区域。
- GC。
- 类加载。
- 异常处理。
- 泛型。
- Stream。

## 2. Java 集合

### 是什么

Java 集合是存储和操作数据的容器。

常见：

- ArrayList。
- LinkedList。
- HashMap。
- HashSet。
- TreeMap。
- PriorityQueue。
- ConcurrentHashMap。

### AI 场景

- List 保存检索结果。
- Map 保存 chunk_id 到 chunk 的映射。
- Set 做去重。
- PriorityQueue 做 TopK。
- ConcurrentHashMap 做本地缓存。

## 3. HashMap

### 是什么

HashMap 是基于哈希表的 key-value 容器。

### 核心机制

Java 8 后结构：

```text
数组 + 链表 + 红黑树
```

当哈希冲突严重，链表长度达到阈值且数组足够大时，链表会转红黑树。

### 面试回答

```text
HashMap 通过 hash 定位桶位，冲突时用链表或红黑树解决。扩容时容量翻倍，并重新分布元素。它线程不安全，并发场景要用 ConcurrentHashMap。
```

## 4. ConcurrentHashMap

### 是什么

ConcurrentHashMap 是线程安全的哈希表。

### 为什么重要

多线程后端服务中，本地缓存、任务状态、连接映射可能需要并发访问。

### 面试重点

- Java 7 分段锁。
- Java 8 CAS + synchronized + Node。
- 读操作大多无锁。

## 5. Java 线程池

### 是什么

线程池复用线程执行任务。

核心参数：

- corePoolSize。
- maximumPoolSize。
- keepAliveTime。
- workQueue。
- threadFactory。
- rejectedExecutionHandler。

### AI 场景

- 文档解析线程池。
- 模型调用线程池。
- Embedding 批处理线程池。

### 面试回答

```text
我会把 CPU 密集和 IO 密集任务隔离线程池。文档解析、OCR 可能偏 CPU 密集，模型调用和向量库查询偏 IO 密集，混在一个线程池里容易互相拖垮。
```

## 6. JVM 内存区域

### 是什么

JVM 运行时内存包括：

- 程序计数器。
- Java 虚拟机栈。
- 本地方法栈。
- 堆。
- 方法区 / 元空间。

### AI 场景

文档解析、批量 chunk、长对话历史都可能造成堆内存压力。

### 常见问题

- OOM。
- Full GC。
- 内存泄露。
- 大对象频繁分配。

## 7. GC

### 是什么

GC 是垃圾回收，自动回收不再使用的对象。

### 常见算法

- 标记清除。
- 标记整理。
- 复制算法。
- 分代收集。

### AI 场景

如果一次性加载大文件或构造巨大 Prompt，会产生大量对象，导致 GC 压力。

- 装饰器。
- 上下文管理器。

## 9. Python async / await

### 是什么

async / await 是 Python 的异步编程机制。

### 适合

- 网络请求。
- 数据库 IO。
- 模型 API 调用。
- 向量库查询。

### 不适合

CPU 密集型任务。CPU 密集任务需要多进程、线程池或专门计算服务。

## 11. 语言选择建议

秋招后端 AI 应用开发推荐：

```text
Java：主后端能力
Python：AI 应用和数据处理能力
Go：了解高并发和云原生加分
C++：如果主语言是 C++，可以走高性能后端、模型网关、推理服务、流式代理、向量检索服务路线
```

面试表达：

```text
我主语言是 Java，能用 Spring Boot 做完整后端系统；AI 应用部分我也能用 Python 快速实现 RAG、文档处理和评测脚本。我的定位是后端工程能力和 AI 应用能力结合。
```

如果你的主语言是 C++，可以这样说：

```text
我主语言是 C++，所以我的 AI 应用项目会强调高性能后端和工程底座：用 C++ 实现 API Server、模型网关、SSE 流式转发、RAG 检索编排和工具调用网关。AI 原型或文档解析可以用 Python 辅助，但核心服务用 C++ 体现网络、并发、内存管理和性能优化能力。
```

现代 C++ 后端不是只能写裸 socket。

```text
裸 socket

  -> C++20 Coroutine 改善异步代码可读性
  -> Drogon 承载完整 Web API
  -> uWebSockets 承载高性能实时连接
```
