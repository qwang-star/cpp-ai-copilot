# 12 两个月冲刺计划

## 总原则

两个月冲刺不能只看资料，要每天产出可面试的东西。

每天固定四件事：

```text
1. 后端基础 1 个专题
2. AI 应用 1 个专题
3. 算法 3 道
4. 项目表达 1 个问题
```

## 第 1 周：后端主干

目标：

- 把普通后端八股恢复到可面试状态。

学习内容：

- TCP / HTTP / HTTPS。
- SSE / WebSocket。
- 进程线程协程。
- IO 模型。
- MySQL 索引。
- MySQL 事务和 MVCC。

产出：

- 整理 20 道后端八股答案。
- 能画普通后端请求链路。

## 第 2 周：中间件与分布式

学习内容：

- Redis 数据结构。
- 缓存穿透、击穿、雪崩。
- 分布式锁。
- MQ 可靠性和幂等。
- 限流、熔断、降级。
- 链路追踪。

产出：

- 能讲清楚“文档上传后异步索引”的设计。
- 写出 MQ 消费幂等方案。

## 第 3 周：大模型基础与模型调用

学习内容：

- Transformer。
- Token 和上下文窗口。
- 模型参数。
- Prompt 基础。
- 结构化输出。
- 流式输出。
- 模型调用封装。

产出：

- 一个聊天接口设计。
- 一个 SSE 流式输出接口设计。
- 一套模型调用日志字段。

## 第 4 周：Embedding 与 RAG

学习内容：

- Embedding。
- 向量数据库。
- metadata。
- 文档解析。
- chunk 切分。
- 向量检索。
- 混合检索。
- Rerank。

产出：

- 画出 RAG 离线和在线链路。
- 整理 20 个 RAG 高频问答。

## 第 5 周：项目工程化

学习内容：

- 文档处理异步化。
- Redis 缓存和限流。
- 模型网关。
- 超时重试。
- 降级兜底。
- 会话管理。

产出：

- 企业知识库问答系统架构图。
- 数据库表设计。
- 接口清单。

## 第 6 周：安全与评测

学习内容：

- Prompt Injection。
- RAG 权限过滤。
- Tool Calling 安全。
- 敏感信息脱敏。
- Golden Dataset。
- Recall@K。
- 答案正确性。
- 幻觉率。

产出：

- 项目安全方案。
- 项目评测方案。
- Bad case 分析模板。

## 第 7 周：项目表达

学习内容：

- STAR 法则。
- 项目三分钟介绍。
- 项目难点包装。
- 项目追问。
- 系统设计题。

产出：

- 3 分钟项目介绍。
- 8 分钟项目介绍。
- 15 分钟项目深挖版。
- 30 个项目追问答案。

## 第 8 周：模拟面试

每天安排：

- 30 分钟算法。
- 40 分钟后端八股。
- 40 分钟 AI 应用题。
- 40 分钟项目追问。
- 20 分钟复盘错题。

产出：

- 高频题答案流畅输出。
- 项目能扛连续追问。
- 算法保持手感。

## 每周自测清单

### 后端

- 能否解释一个请求从浏览器到后端再到数据库的全过程？
- 能否解释 MySQL 索引为什么失效？
- 能否解释 Redis 缓存三大问题？
- 能否解释 MQ 重复消费和幂等？
- 能否解释限流、熔断、降级？

### AI 应用

- 能否完整讲出 RAG 链路？
- 能否说明 chunk 大小怎么选？
- 能否说明 Embedding 和关键词检索区别？
- 能否说明 Tool Calling 的安全边界？
- 能否说明模型调用失败怎么办？
- 能否说明如何评测 AI 应用？

### 项目

- 能否三分钟讲完项目？
- 能否讲出三个真实难点？
- 能否说出具体表设计？
- 能否说出接口设计？
- 能否说出指标和优化？
