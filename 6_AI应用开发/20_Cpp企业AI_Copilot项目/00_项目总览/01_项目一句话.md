# 项目一句话

## 这个项目是什么

```text
C++ 企业级 AI Copilot 是一个面向企业内部员工的 AI 助手系统。
它可以基于公司制度、手册、FAQ 做知识库问答，也可以通过 Tool Calling 查询报销单、请假单、采购单等业务流程。
```

你可以把它想成：

```text
企业内部的 ChatGPT
  +
企业文档搜索引擎
  +
业务流程查询助手
```

## 面试一句话版本

```text
我做的是一个 C++ 实现的企业级 AI 知识库与流程助手系统。核心后端用 C++20 实现，支持文档上传、登录鉴权、SSE 流式问答、MySQL/Redis/MQ 集成。文档入库时通过异步任务完成解析、切分、Embedding 和向量索引；用户提问时先做权限过滤，再进行混合检索和 Rerank，把相关 chunk 拼进 Prompt 调用大模型，并返回带引用的答案。对于报销单、请假单这类实时业务问题，我通过 Tool Calling 调用后端受控接口完成查询。工程上我还做了模型网关、限流、重试、降级、调用日志、token 成本统计和评测集回归。
```
